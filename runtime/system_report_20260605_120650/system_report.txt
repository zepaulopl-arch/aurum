FAILURES FOUND DURING WORK
--------------------------------------------------------------------------------
1. daily_signal first validation timed out because update took longer than the
   command timeout. The orphan process was inspected and later handled.

   PowerShell output. Fixed in scripts/ops_common.ps1.

   sklearn autotune path. Fixed by capping feature history and sampling autotune.

   TradeChat path on every shell invocation. This is external to pyMercator.
